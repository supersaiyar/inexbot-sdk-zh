# 示教器与上位机的通讯连接

### 心跳

- 说明：
- 不管什么通讯模式，都应设计心跳模式，确保通讯状态正常。
- 示教盒定期发送心跳检查包给控制器，并等待控制器回应。客户端收到心跳包后需要回复心跳回应包，包数据为检查包所携带的时间戳。示教盒收到回应包后检查数据是否正确。数据不正确或超时为收到回应，则认为连接可能中断。
- 收到心跳
**命令字：** `0x1200`
- "time":123

```json
{
"time":123
}
```
- 时间戳；int类型
- 回应心跳
**命令字：** `0x1201`
- "time":123

```json
{
"time":123
}
```
### 示教器关机

- 时间戳；int类型
- 说明：
- 拔出示教器发送
- 收到示教器关机
**命令字：** `0x1210`

### 示教器开机

- date:  无
- 说明：
- 插入示教盒恢复通信
- 回应示教器开机
**命令字：** `0x1211`
- date:  无
