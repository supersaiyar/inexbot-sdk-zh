# 操作参数

### 操作参数设置

- 运行参数设置
**命令字：** `0x1300`
- "runModeAutoManBegFlag": false,
- 切到运行模式自动上电； bool类型
- true: 自动上电；false：不自动上电
- "remoteUseReserveModeFlag": false,
- 是否为预约模式；bool类型
- true: 预约模式；false：非预约模式
- "remoteAllowContinueRunFlag": true
- 远程IO断点执行；bool类型
- true: 断电执行；false：断电不执行
- "remoteAllowCurrentRunFlag": false
- 远程IO当前行执行；bool类型
- true: 当前行执行；false：非当前行执行
- "remoteSwitchModeEnable": false,
- 无示教器切远程模式；bool类型
- true: 允许；false：不允许
- "allowReserveRunningPro": false
- 远程IO程序运行中再次预约；bool类型
- true: 允许；false：不允许
- "SetSafeScreenTeachInvalidFlag": false,
- 示教模式安全光幕不生效；bool类型
- true: 不生效；false：生效
- "keepStepRunFlag": 0
- 单步/回零/复位点/运行操作方式；int类型
- 0：单击运行；1：按下运行
- "defaultRunningSpeed": 1
- 运行模式开机默认速度；int类型
- "language":"chinese"
- 设置默认语言；string类型
- 目前支持三种语言，"chinese" ：中文；"english"：英文；"Korean"：韩文
- "accUnitMode":0
- 关节/笛卡尔参数中最大加速度单位；int类型
- 0: 倍数值； 1: 数值
- "craftSelect": "general"
- 工艺选择；string类型
- "general"：通用工艺；"special"：专用工艺；"punch"：冲压工艺；"pallet"：码垛工艺；"weld"：焊接工艺；"cut"：切割工艺
- "base_coord_of_user_coord_": 0
- 用户坐标系基准坐标；bool类型
- 0: 直角坐标系；1: 大地坐标系
- "remote_io_reset_clear_all_reserve_program_status_flag":true
- 远程IO复位清空预约队列 ；bool类型
- true: 清空；false：不清空
- "autoSetUserAndTool":false
- 点位形态/工具手/用户值设置方式；bool类型
- true:自动  false：手动
- "checkServoStatusWordMaxTime": 0
- 伺服响应时间0-100ms；int类型
- 取值范围[0,100]
- "OutageKeepFunctionFlag": true
- 断电保持标志位；bool类型
- true:保持  false：不保持
- "quickInsert":false

```json
{
"runModeAutoManBegFlag":false,            //切到运行模式自动上电
"remoteUseReserveModeFlag": false,         //预约模式
"remoteAllowContinueRunFlag":true,         //远程IO断点执行
"remoteAllowCurrentRunFlag": false         //远程IO当前行执行
"remoteSwitchModeEnable":false,            //无示教器切远程模式
"allowReserveRunningPro": false,          //远程IO程序运行中再次预约
"SetSafeScreenTeachInvalidFlag":false,   //示教模式安全光幕不生效
"keepStepRunFlag": 0 ,1                  //单步/回零/复位点/运行操作方式 0：单击运行，1：按下运行
"defaultRunningSpeed":1                //运行模式开机默认速度
"language":"chinese"                  //"chinese","english","Korean"
"accUnitMode":0,//关节/笛卡尔参数中最大加速度单位 0:倍数值 1：数值
"craftSelect":"general"        //工艺选择：general：通用工艺，special：专用工艺，punch：冲压工艺，pallet：码垛工艺，weld：焊接工艺，cut：切割工艺
"base_coord_of_user_coord_":0//用户坐标系基准坐标 0:直角坐标系  1：大地坐标系
"remote_io_reset_clear_all_reserve_program_status_flag":0//远程IO复位清空预约队列
"autoSetUserAndTool":false,//点位形态/工具手/用户值设置方式  true:自动  false：手动
"checkServoStatusWordMaxTime":0,//伺服响应时间0-100ms
"OutageKeepFunctionFlag":true  //断电保持标志位
"quickInsert":false
}
```
