# 视觉工艺

- 上位机发送，试拍照
**命令字：** `0x8125`
- "cali_camera_angle":false,
- 试拍照记录工件旋转角度；bool类型
- true:记录；false:不记录
- "robot":1
- 当前机器人；int类型；取值范围[1,4]
- "visionNum":0

```json
{
  "cali_camera_angle":false,
  "robot":1,
  "visionNum":0
}
```
- 工艺号；int类型；取值范围[0,98]
- 控制器返回拍照结果
**命令字：** `0x8126`
- "cameraPoint":[0,0,0,0,0,0]
- 相机坐标点；double类型数组
- "protocol"
- "type"
- 数据解析类型；int类型；取值范围[0,2]
- 0:2维；1:2维+高度；2:3维
- "camera_angle":0
- 工件旋转角度；double类型；[-180,180]
- "cameraData":""

```json
{
  "cameraPoint":[0,0,0,0,0,0],
  "protocol":
  {
    "type":0
  },
  "camera_angle":0,
  "cameraData":""
}
```
