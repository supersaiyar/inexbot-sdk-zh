# 寻位跟踪

### 跟踪

#### 跟踪传感器类型

- 上位机发送，设置跟踪传感器类型
**命令字：** `0x8740`
- "fileNum":1
- 跟踪文件号；int类型；取值范围[1,99]
- "robot":1
- 机器人号；int类型；取值范围[1,4]
- "sensorType":0

```json
{
  "fileNum":1,
  "robot":1,
  "sensorType":0
}
```
- 跟踪类型;int类型；取值范围[0,2]
- 0:线激光；1:电弧；2:弧压
- 上位机发送，查询跟踪传感器类型
**命令字：** `0x8741`
- "fileNum":1
- 跟踪文件号；int类型；取值范围[1,99]
- "robot":1

```json
{
  "fileNum":1,
  "robot":1
}
```
- 机器人号；int类型；取值范围[1,4]
- 上位机接收，返回的跟踪传感器类型
**命令字：** `0x8742`

#### 电弧通讯参数

- data：同0x8740
- 上位机发送，设置电弧通讯参数
**命令字：** `0x874d`
- "craftid":1
- 跟踪文件号；int类型；取值范围[1,99]
- "robot":1,
- 机器人号；int类型；取值范围[1,4]
- "sampling":{}
- "dataType":0
- 采样数据类型；int类型；取值范围[0,1]
- 0：电流；1：电压
- "period":20

```json
{
  "craftid":1,
  "robot":1,
  "sampling":
  {
    "dataType":0,
    "period":20
  }
}
```
- 采样周期；int类型；取值范围[0,1000]ms
- 上位机发送，查询电弧通讯参数
**命令字：** `0x874e`
- "craftid":1
- 跟踪文件号；int类型；取值范围[1,99]
- "robot":1,

```json
{
  "craftid":1,
  "robot":1
}
```
- 机器人号；int类型；取值范围[1,4]
- 上位机发送，设置电弧通讯参数
**命令字：** `0x874f`

#### 电弧补偿参数

- data:同0x874d
- 上位机发送，设置电弧跟踪左右补偿参数
**命令字：** `0x8750`
- "craftid":1
- 跟踪文件号；int类型；取值范围[1,99]
- "lateralCompensation":{}
- "accFactor":1.0,
- 补偿加速度倍数；double类型；取值范围[0.1,10]
- 对应当前运动指令加速
- "algorithmType":0
- 偏差提取类型；int类型；取值范围[0,1]
- 0:均值；1:手填值
- "beginCycleNum":4
- 开始采样周期数；int类型；取值范围[1,100]
- "differential":0.0
- 微分系数；double类型；取值范围[0,1]
- "integral":0.00050
- 积分系数；double类型；取值范围[0,1]
- "maxSingleLen":2.0
- 每次最大补偿量；double类型；取值范围[0,10]
- "scale":0.050
- 比例系数；double类型；取值范围[0,1]
- "switchon":true
- 补偿开关；bool类型；true：打开，false：关闭
- "robot":1

```json
{
  "craftid":1,
  "lateralCompensation":
  {
    "accFactor":1.0,
    "algorithmType":0,
    "beginCycleNum":4,
    "differential":0.0,
    "integral":0.00050,
    "maxSingleLen":2.0,
    "scale":0.050,
    "switchon":true
  },
  "robot":1
}
```
- 机器人号；int类型；取值范围[1,4]
- 上位机发送，查询电弧跟踪左右补偿参数
**命令字：** `0x8751`
- "craftid":1
- 跟踪文件号；int类型；取值范围[1,99]
- "robot":1

```json
{
  "craftid":1,
  "robot":1
}
```
- 机器人号；int类型；取值范围[1,4]
- 上位机接收，控制器返回电弧跟踪左右补偿参数
