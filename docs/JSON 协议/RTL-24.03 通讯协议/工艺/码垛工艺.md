# 码垛工艺

## 抓手

- 说明

##### 上位机设置抓手参数

- 抓手设置就是设置码垛过程中用到的工具手。两个吸盘分别取料（一个吸盘吸完后再切换到另一个吸盘取料），那么要设置成两个抓手；两个吸盘同时取料，则设置成一个抓手；如果两个吸盘分别放料（一个吸盘放完后切换到另一个吸盘放料），此时需要设置两个抓手；两个吸盘同时放料，则设置成一个抓手。
**命令字：** `0x8000`
- "robot":1
- 当前机器人；int类型
- 取值范围：[1,4]
- "craftID":1
- 工艺号；int类型
- "pallet":
- "gripperNum":4,
- 抓手数目；int类型
- 取值范围：[1,4]
- "gripper":[1,3,4,1]

```json
{
"robot":1,
"craftID":1,
"pallet":
{
"gripperNum":2,
"gripper":[1,3,4,1]
}
}
```
##### 上位机查询抓手参数

- 抓手工具号；int类型

**命令字：** `0x8001`
- "robot":1

- 取值范围：[1,4]

- true：是；false：否
- "fixedSupHeight": false
- 固定辅助点高度；bool类型
- true：是；false：否

- true：是；false：否

- true：是；false：否
