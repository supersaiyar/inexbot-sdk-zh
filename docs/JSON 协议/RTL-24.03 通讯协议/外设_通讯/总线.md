# 总线

### CanOpen

- 说明
- 上位机查询该硬件的can支持的功能
**命令字：** `0x7000`
- "canPort":1
- can端口号；UInt类型
- "function":0

```json
{
  "canPort":1,
  "function":0
}
```
- 该硬件can支持的功能；int类型，范围[0,3]
- 0表示标准帧，1表示扩展帧，2表示远程帧，3表示过滤id
- 控制器回复上位机查询该硬件的can支持的功能结果
**命令字：** `0x7001`
- "canPort":1
- can端口号；UInt类型
- "support":true

```json
{
  "canPort":1,
  "support":true
}
```
- 该硬件的can是否支持这功能；bool类型
- 上位机设置can参数
**命令字：** `0x7002`
- "canBaud":115200
- can通讯波特率；UInt类型
- "canPort":1
- can通讯端口；UInt类型
- "enableRecvFiltert":true
- 是否开启过滤ID；bool类型
- "frameFormat":0
- 帧格式；int类型，范围[0,1]
- 0表示标准帧，1表示扩展帧
- "recvFilterId":1

```json
{
  "canBaud":115200,
  "canPort":1,
  "enableRecvFiltert":true,
  "frameFormat":0,
  "recvFilterId":1
}
```
- 设置过滤ID；UInt类型
- 上位机请求接收一帧can数据
**命令字：** `0x7003`
- "canPort":1

```json
{
  "canPort":1
}
```
- can通讯端口；UInt类型
- 控制器回复上位机接收一帧can数据结果
**命令字：** `0x7004`
- "canPort":1
- can通讯端口；UInt类型
- "data":[10]
- 接收到的数据；UChar类型
- 长度为"Length"节点的值
- "Length":1
- 接收到的数据长度；UInt类型
- "recvCanID":1

```json
{
  "canPort":1,
  "data":[10],
  "Length":1,
  "recvCanID":1
}
```
- 接收到的帧ID；UInt类型
- 上位机请求发送一帧数据
**命令字：** `0x7005`
- "canPort":1
- can通讯端口；UInt类型
- "data":[10]
- 需要发送的数据；UChar类型
- "sendCanID":1
- 发送帧的ID；UInt类型
- "sendLen":1
- 发送的数据长度；UInt类型，范围[1,8]
- "useRemoteFrame":true

```json
{
  "canPort":1,
  "data":[10],
  "sendCanID":1,
  "sendLen":1,
  "useRemoteFrame":true
}
```
### ENI

- 是否使用远程帧；bool类型
- 说明
- 上位机查询ENI文件列表
**命令字：** `0x7010`
- "num":10

```json
{
  "num":10
}
```
- 获取ENI文件数目；int类型，范围1,10
- 1时获取当前使用的ENI文件，10表示获取所有ENI文件
- 控制器回复上位机ENI文件列表
**命令字：** `0x7011`
- "ENIfilelist":["INEXBOT-IO-R4-1.xml","DST_X503-1.json","DST_X503-1.xml","KunWeiTech_FT_Sensors-1.xml","P140000107-1.0.1.1-ECXML-1.xml"]
- 控制器ENI文件列表；string类型
- "ENIfilenum":10
- ENI文件数目；int类型
- "absolutepath":"eni/"

```json
{
  "ENIfilelist":["INEXBOT-IO-R4-1.xml","DST_X503-1.json","DST_X503-1.xml","KunWeiTech_FT_Sensors-1.xml","P140000107-1.0.1.1-ECXML-1.xml"],
  "ENIfilenum":10,
  "absolutepath":"eni/"
}
```
- 控制器中ENI文件所在文件夹
- 上位机查询ENI参数

```json
{}
```
**命令字：** `0x7012`
- 控制器回复上位机ENI参数
**命令字：** `0x7013`
- "ENIName":"eni-RC-6-mecat-1-1000.xml"
- ENI文件名称；string类型
- "isHaveENI":1
```json
{
  "ENIName":"eni-RC-6-mecat-1-1000.xml",
  "isHaveENI":1
}
```

{
