# IO

### 状态提示

- 说明
- 提示端口范围最大均为当前绑定的IO数量
- 输出方式范围均为[0,2]，0为0，1为1，2为闪烁

- 输出端口复位值；int类型，范围[0,1]

- 上位机查询IO功能状态

- 控制器回复上位机IO功能状态

### IO配置
- IO状态类型；int类型，范围[1,4]
- 1表示数字输入，2表示数字输出，3表示模拟输入，4表示模拟输出
- 说明
- 串口模拟IO参数(若EtherCAT IO板有模拟IO 则该串口模拟IO将无效)
- 上位机设置IO配置参数
**命令字：** `0x7130`
- "serialAnalog":
- 串行模拟参数；
- "baudRate":115200
- 波特率；int类型
- "port":2
- 端口号；int类型，范围(0,65535]
- "type":"SUPER_ANAIO"
- 模拟类型；string类型
- "DAC_ANAIO"表示"DAC模拟IO板"，"SUPER_ANAIO"表示"格控模拟IO板"
- "simuNum":2
```json
{
  "serialAnalog":
  {
    "baudRate":115200,
    "port":2,
    "type":"SUPER_ANAIO"
  },
  "simuNum":2
}
```
- 虚拟IO数量；int类型，范围[0,4]
- 上位机查询IO配置参数
```json
{}
```
