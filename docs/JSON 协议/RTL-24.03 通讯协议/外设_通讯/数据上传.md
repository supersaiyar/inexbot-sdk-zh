# 数据上传

### 数据上传

- 说明
- 上位机设置数据传输开关
**命令字：** `0x7330`
- "enable":true
- 数据传输开关；bool类型
- "robot":1

```json
{
  "enable":true,
  "robot":1
}
```
- 当前机器人号
- 上位机查询数据传输开关
**命令字：** `0x7331`
- "robot":1

```json
{
  "robot":1
}
```
- 当前机器人号
- 控制器回复上位机数据传输开关
**命令字：** `0x7332`
- "enable":false
- 数据传输开关；bool类型
- "robot":1

```json
{
  "enable":false,
  "robot":1
}
```
- 当前机器人号
- 上位机设置数据上传参数
**命令字：** `0x7333`
- "dataCollectTime":1.0
- 数据采集周期；int类型，范围[0,999999]s
- "dataUploadTime":30.0
- 数据上传周期；int类型，范围[0,999999]s
- "fileSuffix":"csv"
- 文件格式；string类型，取值"csv","txt"
- "filepath":""
- 文件路径；string类型
- "password":""
- 密码；string类型
- "robot":1
- 机器人号；int类型，范围[1,4]
- "sendInfoFileFlag":false
- 是否发送说明文档；bool类型
- "serverIP":"192.168.1.14"
- 服务器IP；string类型
- "serverPort":21
- 端口号；int类型，范围(0,65535]
- "uploadMode":"FTP"
- 上传方式；string类型，默认"FTP"
- "username":""

```json
{
  "dataCollectTime":1.0,
  "dataUploadTime":30.0,
  "fileSuffix":"csv",
  "filepath":"",
  "password":"",
  "robot":1,
  "sendInfoFileFlag":false,
  "serverIP":"192.168.1.14",
  "serverPort":21,
  "uploadMode":"FTP",
  "username":""
}
```

### 数据上传

- 用户名；string类型
- 说明
- 上位机设置与查询分四次进行
- 上位机设置数据格式
**命令字：** `0x7336`
- "format":"Robot-R1_ $Y%-$m%-$d%_ $H%-$M%-$S%"
- 数据格式；string类型
- 该格式表示 Robot-R1_年-月-日_时-分-秒
- 详细说明参考数据格式
- "robot":1
- 当前机器人号；int类型，范围[1,4]
- "type":1

```json
{
  "format":"Robot-R1_ $Y%-$m%-$d%_ $H%-$M%-$S%",
  "robot":1,
  "type":1
}
```
- 当前设置的格式类型；int范围，范围[1,4]
- 1表示说明文档文件名，2表示说明文档文件内容，3表示数据文档文件名，4表示数据文档文件内容
- 上位机查询数据格式
**命令字：** `0x7337`
- "robot":1
- 当前机器人号；int类型，范围[1,4]
- "type":1

```json
{
  "robot":1,
  "type":1
}
```
- 当前设置的格式类型；int范围，范围[1,4]
- 1表示说明文档文件名，2表示说明文档文件内容，3表示数据文档文件名，4表示数据文档文件内容
- 控制器回复上位机数据格式
**命令字：** `0x7338`
- "format":"Robot-R1_ $Y%-$m%-$d%_ $H%-$M%-$S%"
- 数据格式；string类型
- 该格式表示 Robot-R1_年-月-日_时-分-秒
- "robot":1
- 当前机器人号；int类型，范围[1,4]
- "type":1

```json
{
  "format":"Robot-R1_ $Y%-$m%-$d%_ $H%-$M%-$S%",
  "robot":1,
  "type":1
}
```
- 当前设置的格式类型；int范围，范围[1,4]
- 1表示说明文档文件名，2表示说明文档文件内容，3表示数据文档文件名，4表示数据文档文件内容
