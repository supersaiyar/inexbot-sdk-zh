# 其他外设

### 编码器

- 说明
- 上位机设置外部轴编码器溢值参数
**命令字：** `0x7200`
- "externalAxis":
- 外部轴编码器最大值最小值参数列表，长度为当前外部轴轴数
- "encode":
- 编码器最大值最小值参数列表，长度为该轴从动轴数目+1
- "max":"100.0"
- 编码器最大值；double类型，范围[-2^32,2^32]
- "min":"-100.0"
- 编码器最小值；double类型，范围[-2^32,2^32]
- "externalGroupNum":1
- 要设置参数的轴组号；int类型，范围[1,3]
- "robot":2

```json
{
  "externalAxis":
  [
    {
      "encode":
      [
        {
          "max":"100.0",
          "min":"-100.0"
        }
      ]
    }
  ],
  "externalGroupNum":1,
  "robot":2
}
```
- 当前机器人号；int类型，范围[1,4]
- 上位机查询外部轴编码器溢值参数
**命令字：** `0x7201`
- "externalGroupNum":1
- 要查询参数的轴组号；int类型，范围[1,3]
- "robot":2

```json
{
  "externalGroupNum":1,
  "robot":2
}
```
- 当前机器人号；int类型，范围[1,4]
- 控制器回复上位机外部轴编码器溢值参数
**命令字：** `0x7202`
- "externalAxis":
- 外部轴编码器最大值最小值参数列表，长度为当前外部轴轴数
- "enable":0
- 编码器多圈值溢出计数功能；int类型，范围[0,1]
- 0表示未开启，1表示已开始
- "encode":
- 编码器最大值最小值参数列表，长度为该轴从动轴数目+1
- "max":"100.0"
- 编码器最大值；double类型，范围[-2^32,2^32]
- "min":"-100.0"
- 编码器最小值；double类型，范围[-2^32,2^32]
- "externalGroupNum":1
- 轴组号；int类型，范围[1,3]
- "robot":2

```json
{
  "externalAxis":
  [
    {
      "enable":0,
      "encode":
      [
        {
          "max":"100.0",
          "min":"-100.0"
        }
      ]
    }
  ],
  "externalGroupNum":1,
  "robot":2
}
```

- 硬盘预警时间；int类型，范围(0,9999)h

### PWM

- 编码器值；int类型
- 说明
- 上位机设置PWM参数
**命令字：** `0x7260`
- "duty":50
- pwm占空比；double类型
- "freq":1000
- pwm频率；double类型
- "pwmOutTimeMs":1000
- pwm输出时间；UInt类型，范围[1000-5000]ms
- 输入0的时候则一直输出pwm
- "pwmPort":1

```json
{
  "duty":50,
  "freq":1000,
  "pwmOutTimeMs":1000,
  "pwmPort":1
}
```
- pwm端口；UInt类型
- 上位机查询PWM参数
**命令字：** `0x7261`
- "pwmPort":1

```json
{
  "pwmPort":1
}
```
- pwm端口；UInt类型
- 控制器回复上位机PWM参数
**命令字：** `0x7262`
- "duty":50
- pwm占空比；double类型
- "freq":1000
- pwm频率；double类型
- "pwmPort":1

```json
{
  "duty":50,
  "freq":1000,
  "pwmPort":1
}
```
- pwm端口；UInt类型
- 控制器回复上位机输出时间结果
**命令字：** `0x7263`
- "ok":true

```json
{
  "ok":true
}
```
- 输出时间结果；bool类型
