```json
{
  "robot": 1
}
```
### 运行程序

- 选择机器人；int 类型；取值范围 [1,4]
- 说明：
- 对程序操作
- 上位机给控制器发送开始运行程序的时候，发送下面命令：
**命令字：** `0x5601`
- "robot":1
- 表示要操作的机器人编号；int类型，取值范围[1,4]
- "jobname":"TTT"
- 程序名；string类型
- "suffixname":".JBR"
- 程序名后缀名称
- 取值：.JBR主程序  .JBP后台局部程序  .JBPG后台全局程序
- "stepRunSum":2
- 一共多少步，可选
- "callerName":"tttt"
- 访问者名字 ，可选
- "stepRun":false,
- 可选
- "index":0,
- 可选
- "noFileRun":false,
- 可选
- "notJudgeSafePos":false,
- 可选
- "line":1
- 程序指令行数；int类型
- 程序开始运行的行号
- "globalStart":false
- 是否为多机模式；bool类型
- "continueRun":0
- 是否为继续运行；int类型；取值范围[0,1]
- 1:继续运行,0:不继续运行
- "startOver":0,

```json
{
    "robot": 1,
    "jobname": "TTT",
    "suffixname": ".JBR",
    "stepRunSum": 2,
    "callerName": "tttt",
    "stepRun": false,
    "index": 0,
    "noFileRun": false,
    "notJudgeSafePos": false,
    "line": 1,
    "globalStart": false,
    "continueRun": 0,
    "startOver": 0,
}
```
注：可选的不使用无影响

- 是否为重头运行；int类型；取值范围[0,1]
- 1:重头运行，0:继续运行
- 上位机下发停止正在运行的程序
**命令字：** `0x5602`

```json
{
    "robot":1
}
```
### 程序执行状态设置

- "robot":1
- 表示要操作的机器人编号；int类型，取值范围[1,4]
- 说明：
- 上位机需要改变当前控制器程序执行模式的时候，发送下面命令
**命令字：** `0x5603`
- "robot": 1
- 选择机器人；int 类型；取值范围 [1，4]
- "type":5
- 程序执行状态；int 类型；取值范围[0,2]||5
- 0:单步运行；1:暂停；2:停止但是不下电；5:停止但不下电(停止优先使用5)

```json
{
  "type": 5,
  "robot": 1
}
```
### 使用无文件运行开关

- 注意：对于单步：没有全部机器人；对于暂停和停止，只有多机模式下有全部机器人，此时将"robot"节点设置成0
- 上位机使用无文件运行开关的时候，发送下面命令：
**命令字：** `0x5604`
- "robot": 1

```json
{
  "robot": 1
}
```

### 循环次数

- 是否反向顺序运行；bool 类型
- 上位机设置机器人循环次数时，发送
**命令字：** `0x5620`
- "robot": 1

- "index":

```json
  "robot": 1,
  "index": 1

```

- 上位机查询机器人循环次数时，发送
**命令字：** `0x5621`
- "robot": 1

```json
{
  "robot": 1
}
```
- 选择机器人；int 类型；取值范围 [1，4]
- 控制器回复查询循环次数时，返回
**命令字：** `0x5622`
- "robot": 1
- 选择机器人；int 类型，取值范围 [1，4]
- "index":
- 总循环次数，0：表示无限循环；int 类型
- "count":


### 全局线程

- 当前已运行次数；int 类型
- 上位机下发，全局线程自启动设置
**命令字：** `0x5630`
- "robot": 1
- 选择机器人；int 类型，取值范围 [1，4]
- "name":"Q1"

```json
{
  "robot":1,
  "name":"Q1"
}
```
- 全局后台程序名；string类型
- 上位机下发，全局线程自启动查询
**命令字：** `0x5631`
- "robot": 1

```json
{
  "robot":1
}
```
- 选择机器人；int 类型，取值范围 [1，4]
- 上位机接收，全局线程自启动接收
**命令字：** `0x5632`
- "robot": 1
- 选择机器人；int 类型，取值范围 [1，4]
- "name":"Q1"

```json
{
  "robot":1,
  "name":"Q1"
}
```
- 全局后台程序名；string类型
- 上位机下发，全局线程启动控制
**命令字：** `0x5633`
- "robot": 1
- 选择机器人；int 类型，取值范围 [1，4]
- "type":1

```json
{
  "robot":1,
  "type":1
}
```
- 启动控制；int类型；取值范围[0,1]
- 1:开始；0:表示停止
- 上位机下发，全局线程启动状态查询
**命令字：** `0x5634`
- "robot": 1
- 选择机器人；int 类型，取值范围 [1，4]
- "hasRun":true
- 自启状态；bool类型
- true：启动；false：停止
- "runName":1

```json

"hasRun":true,
"runName":1
}
```
- 获取全局后台运行状态
- 上位机下发，全局线程启动状态查询
**命令字：** `0x5634`
- "robot": 1
- 选择机器人；int 类型，取值范围 [1，4]
- "hasRun":true

```json
{
"robot":1,
"hasRun":true,
}
```
查询全局后台自启状态

*0x50c5  GLOBAL_PTHREAD_RUNSTATUS_INQUIRE

```json
{
“robot”:1    //当前机器人号
“hasRun":true  //自启状态true表示启动false表示停止
"runName":1   //1：获取全局后台运行状态
}
```

控制器回复

*0x50c6  GLOBAL_PTHREAD_RUNSTATUS_RESPOND

```json
{
“robot”:1    //当前机器人号
“hasRun":false,true  //自启状态true表示启动false表示停止
}
```

### 追加运动模式

- 自启状态；bool类型
- true：启动；false：停止
- 说明：
- 追加运动模式是在不使用程序的条件下运行程序
- 使用方法：
- 打开追加运动模式
- 追加运动模式插入指令向量
- 追加运动模式设置
**命令字：** `0x5640`
- "robot":1
- 表示要操作的机器人编号；int类型，取值范围[1,4]
- "open":true

```json
{
  "robot":1,
  "open":true
}
```
- 是否开启追加运动模式；bool类型
- true:打开；false:关闭
- 上位机查询追加运动模式
**命令字：** `0x5641`
- "robot":1

```json
{
  "robot":1
}
```
- 表示要操作的机器人编号；int类型，取值范围[1,4]
- 控制器收到查询命令，发送下面命令
**命令字：** `0x5642`
- "robot":1
- 表示要操作的机器人编号；int类型，取值范围[1,4]
- "open":true

```json
{
  "robot":1,
  "open":true
}
```
- 是否开启追加运动模式；bool类型
- true:打开；false:关闭
- 追加运动插入指令向量
**命令字：** `0x5643`
- "robot":1
- 表示要操作的机器人编号；int类型，取值范围[1,4]
- "isContinue":false
- true:只接收并存储pos；false：接收pos，并使得机器人执行所有pos
- "data":[]
- 指令json列表
- cmd->toJson()

```json
{
  "data":
  [
  {cmd->toJson()},
  {cmd->toJson()},
  {cmd->toJson()}
  ],
  "isContinue":false,
  "robot":1
}
```
- 指令详情；请看指令页面
- 追加运动模式挂起(暂停）
**命令字：** `0x5644`
- "robot":1

```json
{
  "robot":1
}
```
- 表示要操作的机器人编号；int类型，取值范围[1,4]
- 追加运动模式启动
**命令字：** `0x5645`
- "robot":1

```json

  "robot":1
}
```

- 表示要操作的机器人编号；int类型，取值范围[1,4]
- 追加运动模式停止
**命令字：** `0x5646`
- "robot":0

```json
{
  "robot":0
}
```
