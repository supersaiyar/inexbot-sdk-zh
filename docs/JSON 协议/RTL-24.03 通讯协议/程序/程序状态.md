# 程序状态

### 程序行号

- 当控制器开始运行一条指令时，向上位机发送
**命令字：** `0x5400`
- "robot": 1
- 选择机器人；int 类型；取值范围 [1，4]
- "jobname":"Q1"
- 作业文件名称；string 类型
- "num":3

```json
{
  "robot": 1,
  "jobname": "Q1",
  "num": 3
}
```
### 当控制器执行CRAFTLINE指令时，向示教器发送跳转：

- 已运行指令数量；int 类型
- 上位机收到数据
**命令字：** `0x5401`
- "robot": 1
- 选择机器人；int 类型；取值范围 [1，4]
- "num":

```json
{
  "robot": 1,
  "num": 2
}
```
### 当示教器与控制器建立网络连接时，上位机查询当前运行到哪一行：

- 行号；int 类型
- 上位机发送数据
**命令字：** `0x5402`
- "robot": 1

```json
{
  "robot": 1
}
```
- 选择机器人；int 类型；取值范围 [1，4]
- 控制器回复查询结果
**命令字：** `0x5403`
- "robot": 1
- 选择机器人；int 类型；取值范围 [1，4]
- "jobname":
- 作业文件名称；string 类型
- "num":

```json
{
  "robot": 1,
  "jobname": "Q1",
  "num": 3
}
```
### 程序运行状态

- 已运行指令数量；int 类型
- 说明：
- 程序状态包括暂停，运行，停止
- 上位机查询程序运行状态时，发送
**命令字：** `0x5410`
- "robot": 1

```json
{
  "robot": 1
}
```
- 选择机器人；int 类型；取值范围 [1，4]
- 控制器回复查询结果
**命令字：** `0x5411`
- "robot": 1
- 选择机器人；int 类型；取值范围 [1，4]
- "status":

```json
{
  "robot": 1,
  "status": 0
}
```
- 程序运行状态；int 类型；取值范围[0,2]
- 0:停止；1:暂停；2:运行
- 上位机发送查询机器人运行状态
**命令字：** `0x5412`
- "robot": 1
- 选择机器人；int 类型；取值范围 [1，4]
- "jobfilename":

```json
{
  "robot": 1,
  "jobfilename": ""
}
```
- 作业文件名；string 类型
- 控制器回复机器人运行状态
**命令字：** `0x5413`
- "robot": 1
- 选择机器人；int 类型；取值范围 [1，4]
- "status":
- 0表示停止，1表示暂停，2表示运行；int 类型
- "continueRun":
- 是否从当前行运行，int 类型；取值范围[0,1]
- 0-从头运行，1-从当前行运行
- "currentRun":
- 当前是否运行；bool 类型
- "mainProgramRun":

```json
{
  "robot": 1,
  "status": 0,
  "continueRun": 0,
  "currentRun": false,
  "mainProgramRun": 0
}
```
- 主程序是否运行；int 类型
- 上位机发送当前行运行设置
**命令字：** `0x5414`
- "robot": 1
- 选择机器人；int 类型；取值范围 [1，4]
- "jobfilename":
- 作业文件名；string 类型
- "num":

```json
{
  "robot": 1,
  "jobfilename": "",
  "num": 1
}
```
- 行号；int 类型
- 上位机发送查询机器人运行状态
**命令字：** `0x5415`
- "robot": 1

```json
{
  "robot": 1
}
```
- 选择机器人；int 类型；取值范围 [1，4]
- 控制器回复机器人运行状态
**命令字：** `0x5416`
- "robot": 1
- 选择机器人；int 类型；取值范围 [1，4]
- "jobfilename":
- 作业文件名；string 类型
- "num":

```json
{
  "robot": 1,
  "jobfilename": "",
  "num": 1
}
```
- 行号；int 类型
- 上位机发送清除运行状态（在导入程序成功后，上位机会发送此协议请求重置程序运行状态）
**命令字：** `0x5417`
- date：无
- 控制器回复数据
**命令字：** `0x5418`
- date：无
- 上位机查询当前运行队列剩余的指令数量
**命令字：** `0x5419`
- "robot":1

```json
{
  "robot":1
}
```
- 当前机器人；int类型；取值范围[1,4]
- 控制器回复查询当前运动队列剩余的指令数量
**命令字：** `0x541a`
- "sum":4

```json
{
  "sum":4
}
```
- 运动队列的行号；int类型
