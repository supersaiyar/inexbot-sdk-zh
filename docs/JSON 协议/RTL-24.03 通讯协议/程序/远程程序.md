# 远程程序

### 预约执行状态

- 远程模式界面，上位机查询预约执行状态时，发送
**命令字：** `0x5500`
- "robot": 1

```json
{
    "robot": 1
}
```
- 选择机器人；int 类型；取值范围 [1，4]
- 控制器回复预约执行状态查询结果
**命令字：** `0x5501`
- "robot": 1
- 选择机器人；int 类型；取值范围 [1，4]
- "queue":1
- 状态；int类型
- 0-无预约，1-预约中，2-运行中，3-已预约，4-程序暂停
- "station":1
- 工位编号；int 类型
- "name":
- 程序名称；string类型
- "runtime":
- 运行时间；int 类型
- "times":
- 运行次数；int 类型
- "count":
- 已运行次数；int 类型
- "status":

```json
{
    "robot": 1,
    "current": {
        "station": 1,
        "name": "Q1",
        "runtime": 1,
        "times": 3,
        "count": 1,
        "status": 2
    },
    "queue": [
        {
            "station": 1,
            "name": "Q4",
            "times": 2,
            "count": 0,
            "status": 1
        },
        {
            "station": 1,
            "name": "Q1",
            "times": 6,
            "count": 0,
            "status": 1
        },
        {
            "station": 1,
            "name": "Q3",
            "times": 1,
            "count": 0,
            "status": 1
        },
        {
            "station": 1,
            "name": "",
            "times": 1,
            "count": 0,
            "status": 0
        },
        {
            "station": 1,
            "name": "",
            "times": 2,
            "count": 0,
            "status": 0
        }
    ]
}
```

- IO触发端口绑定，0：表示“无”，1：表示“端口1”

### 远程模式连接状态

- 对应每个远程 IO 程序的输出端口值；int 类型
- 查询Modbus,外部Io连接状态时，发送
**命令字：** `0x5560`
- “date”: 无
- 收到查询连接状态时，返回
**命令字：** `0x5561`
- "ModbusConnect":false
- Bool 类型
- modbus连接状态，:false：未连接，true：已连接
- "ExternIOConnect":true
- Bool 类型
- 外部IO连接状态，:false：未连接，true：已连接
- "type":2

```json
{
"ModbusConnect":false,
"ExternIOConnect":true,
"type":2
}
```
- int 类型
- 0:NAN,  1:RTU,  2:TCP
- 控制器发送，是否通过io修改了作业文件
**命令字：** `0x5562`
- "isModify":bool

```json
{
"isModify":bool
}
```
注：外部按键触发修改作业文件的时候，控制器发送给上位机

- 是否通过io修改了作业文件；bool类型
- true：是；false：否
