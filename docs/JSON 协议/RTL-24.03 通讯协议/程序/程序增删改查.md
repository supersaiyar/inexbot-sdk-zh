# 程序增删改查

### 程序删除

- 说明：
- 在删除程序时使用
- 上位机下发删除程序
**命令字：** `0x5000`
- "robot":1
- 表示要操作的机器人编号；int类型，取值范围[1,4]
- "jobfilename":"QQQQ"
- 作业文件名称；string类型
- "suffixname":".JBR"

```json
{
  "robot":1,
  "jobfilename":"QQQQ",
  "suffixname":".JBR"
}
```
### 程序重命名

- 作业文件后缀名；string类型
- ".JBR"主程序，".JBP"后台局部程序，".JBPG"后台全局程序
- 说明：
- 将原有程序的名字更改
- 上位机下发重命名程序
**命令字：** `0x5001`
- "robot":1
- 表示要操作的机器人编号；int类型，取值范围[1,4]
- "lastname":"QQQQ"
- 修改前程序名；string类型
- "newname":"WWWW"
- 修改后程序名；string类型
- "suffixname":".JBR"
- 程序后缀名；string类型

```json
{
  "robot":1
  "lastname":"QQQQ"
  "newname":"WWWW"
  "suffixname":".JBR"
}
```
### 程序创建

- ".JBR"主程序，".JBP"后台局部程序，".JBPG"后台全局程序
- 说明
- 程序的新建需要在示教模式下
- 程序的运行需要再运行模式下，并且伺服就绪，上电没有报错
- 上位机下发新建程序
**命令字：** `0x5002`
- "robot":1
- 表示要操作的机器人编号；int类型，取值范围[1,4]
- "jobName":"TTT"
- 新建程序名；string类型
- 程序命名首字母不得为数字
- "suffix":".JBR"
- ".JBR主程序"，".JBP"后台局部程序，".JBPG"后台全局程序
- "time":"2025.03.11 20:28:23"

```json
{
  "jobName":"TTT",
  "robot":1,
  "suffixname":".JBR",
  "time":"2025.03.11 20:28:23"
}
```
- 表示插入指令的时间；string类型
- 控制器发送给示教器外部按键触发的新建作业
- 0x5004
- "jobName":"文件1"

```json
{
    "jobName":"文件1",
}
```
### 文件

- 需要新建的作业名称；string类型
- 说明
- 0x5010查询程序文件列表，若成功将会返回0x5011、0x5012、0x5013三个命令字
- 使用时，在接收到0x5011后，开启计时，设定5秒，作用为判断下载文件过程中是否等待超时
- 返回0x5012时，每个机器人会单独发送，并且每次发送文件数量为五个
- 文件发送完成后，返回0x5013作为发送完成的标志
- 上位机下发查询程序文件列表
**命令字：** `0x5010`
- 无节点
- 获取所有机器人的程序文件列表
- 控制器返回程序文件总数
**命令字：** `0x5011`
- "absolutepath":{"/job/R1/","/job/R2/","/job/R3/","/job/R4/"}
- 程序文件的文件夹列表；string类型
- 当只有一个机器人时，则第二、三、四个字符串为""
- "jobfilenum":[4,1,1,1]
- 程序文件夹中包含的文件个数；int数组类型；
- 分别是机器人1,2,3,4；有四个int类型元素；数量包含复位程序
- 当只有一个机器人时，则第二、三、四个元素值为0
- 备注：

```json
{
  "absolutepath":{"/job/R1/","/job/R2/","/job/R3/","/job/R4/"},
  "jobfilenum":{4,1,1,1}
}
```
- 复位程序：当点击复位时执行复位程序或者移动到复位点，具体操作看用户设置
- 控制器主动向上位机发送指令
**命令字：** `0x5012`
- "robot":1
- 表示要操作的机器人编号；int类型，取值范围[1,4]
- 向上位机回复时每个机器人分开发送
- "listnum":2
- 发送的程序文件数量；int类型；[0,5]
- 单次最多发送5个，数量过多时会发送多次
- "jobfilelist":[]
- 程序文件数组；object类型数组
- 元素个数为"listnum"节点的值
- "name":"xxx.JBR"
- 程序文件名字；string类型
- "MD5":"5b172a7a37a07aa6ccec1c60cfd5db7d"
- MD5值作用是判断文件是否相同；string类型
- 备注：

```json
{
    "robot": 1,
    "listnum": 2,
    "jobfilelist":
    [
        {
            "name": "RobotResetProgram.ResetPro",
            "MD5": "e4b21ca518cd2d2eecd4b3f463c02ab1"
        },
        {
            "name": "yyy.JBR",
            "MD5": "5b172a7a37a07aa6ccec1c60cfd5db7d"
        }
    ]
}
```
- RobotResetProgram.ResetPro为复位程序
- 控制器返回程序文件列表查询结束的标识
**命令字：** `0x5013`

### lua文件

- 无节点
- 标志着程序文件列表查询结束
- 上位机发送查询lua文件列表
- 0x5020
- 无节点
- 控制器返回lua文件列表
- 0x5021
- "absolutepath":"lua/"
- lua脚本所在目录；string类型
- "sum":2
- 列表总数；int类型
- "scriptlist":["ww.lua","ee.lua"]

```json
{
  "absolutepath":"lua/",
  "sum":2,
  "scriptlist":["ww.lua","ee.lua"]
}
```
- lua文件列表；string数组类型
