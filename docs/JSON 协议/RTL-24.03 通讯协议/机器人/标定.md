# 标定

### 四点标定

- 说明
- 机器人末梢移动到并标定四个点，计算后可得杆长度和轴零点偏移，若认可计算结果，可将计算结果填入DH参数
- 上位机四点标定计算请求
**命令字：** `0x3700`
- "ParamOf4Points":[85.0,70.0]

```json
{
  "ParamOf4Points":[85.0,70.0]
}
```
- L1和L2长度；double类型
- 两参数分别为机器人末梢从标记点A移动到标记点C时X的移动距离和Y的移动距离
- 上位机N号点标定请求
**命令字：** `0x3701`
- "pointNum":0
- 标定点编号；int类型，范围[0,3]
- "status":1

```json
{
  "pointNum":0,
  "status":1
}
```
- 标定状态；int类型，范围[0,1]
- 0表示取消标定，1表示标定
- 控制器回复上位机N号点标定位置信息
**命令字：** `0x3702`
- "pointNum":0
- 标定点编号；int类型，范围[0,3]
- "pos":[0.0,0.0,0.0,0.0]

```json
{
  "pointNum":0,
  "pos":[0.0,0.0,0.0,0.0]
}
```
- 位置信息列表；double类型
- 长度为4
- 控制器回复上位机标定计算结果
**命令字：** `0x3703`
- "dtheta":[-1.0,-1.0]
- 轴零点偏移列表；double类型
- 长度为2
- 值全为-1表示计算失败
- "length":[-1.0,-1.0]
- 杆长度列表；double类型
- 长度为2
- 值全为-1表示计算失败
- "result":false

```json
{
  "dtheta":[-1.0,-1.0],
  "length":[-1.0,-1.0],
  "result":false
}
```
- 是否计算成功；bool类型
- 上位机请求将计算结果填入DH参数

```json
{}
```
**命令字：** `0x3704`
- 控制器回复上位机填入DH参数结果
**命令字：** `0x3705`
- "apply":true
```json
{
  "apply":true
}
```
- 计算结果填入是否成功；bool类型
- 上位机查询标定信息
**命令字：** `0x3706`
- "dtheta":[null,null]
- 轴零点偏移列表；double类型
- 长度为2
- null表示没有计算结果
- "length":[null,null]
- 杆长度列表；double类型
- 长度为2
- null表示没有计算结果
- "result":false
- 是否计算成功；bool类型
- "status":[0,1,0,1]
```json
{
  "dtheta":[null,null],
  "length":[null,null],
  "result":false,
  "status":[0,1,0,1]
}
```
### 20点标定
- 点位标定状态；int类型，范围[0,1]
- 0表示未标定，1表示已标定
- 说明
- 一种用于校准机器人的方法，通过在特定的20个点上进行标定，以确保机器人的准确性和稳定性
- 上位机标定计算请求
**命令字：** `0x3710`
- "calibrationType":20
- 标定点方法；int类型，范围2,12,15,20,21
- 2表示2点法，12表示12点法，15表示15点法，20表示20点法有校准零点，21表示21点法无校准零点
- "noCalZero": false
- 是否为校准零点；bool类型
- "robot": 1
- 当前机器人号；int 类型，范围 [1,4]
- "tool":1
```json
{
  "calibrationType": 7,
  "noCalZero": false,
  "robot": 1,
  "tool": 1
}
```
- 当前工具手编号；int 类型，范围[1,999]
- 上位机标定N号点请求
**命令字：** `0x3711`
- "calibrationType":20
- 标定点方法；int类型，范围2,12,15,20,21
