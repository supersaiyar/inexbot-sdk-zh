# 跟随误差&电机过载保护

### 最大跟随误差上限

- 说明
- 上位机设置最大跟随误差参数上限
**命令字：** `0x3860`
- "maxDynamicDeviations":[5000000,5000000,5000000,5000000,5000000,5000000]
- 最大动态误差；int类型，范围[1,2*10^7]‱°
- "maxStaticDeviations":[5000000,5000000,5000000,5000000,5000000,5000000]

```json
{
  "maxDynamicDeviations":[5000000,5000000,5000000,5000000,5000000,5000000],
  "maxStaticDeviations":[5000000,5000000,5000000,5000000,5000000,5000000]
}
```
- 最大静态误差；int类型，范围[1,2*10^7]‱°
- 上位机查询最大跟随误差参数上限
**命令字：** `0x3861`
- "robot":1

```json
{
  "robot":1
}
```
- 当前机器人号；int类型，范围[1,4]
- 控制器回复上位机最大跟随误差参数上限
**命令字：** `0x3862`
- "maxDynamicDeviations":[5000000,5000000,5000000,5000000,5000000,5000000]
- 最大动态误差；int类型，范围[1,2*10^7]‱°
- "maxStaticDeviations":[5000000,5000000,5000000,5000000,5000000,5000000]
- 最大静态误差；int类型，范围[1,2*10^7]‱°
- "robot":1

```json
{
  "maxDynamicDeviations":[5000000,5000000,5000000,5000000,5000000,5000000],
  "maxStaticDeviations":[5000000,5000000,5000000,5000000,5000000,5000000],
  "robot":1

### 电机过载保护

- 当前机器人号；int类型，范围[1,4]

- 上位机设置电机过载保护参数
**命令字：** `0x3880`
- "enable":false
- 电机过载保护使能；bool类型
- "minLimiteTime":10
- 电机过载保护最小确认时间；int类型，范围[10,9999]‰
- "robot":1
- 当前机器人号；int类型，范围[1,4]
- "robotLimit":[999,999,999,999,999,999]

```json
{
  "enable":false,
  "minLimiteTime":10,
  "robot":1,
  "robotLimit":[999,999,999,999,999,999]
}
```

- 上位机查询电机过载保护参数
**命令字：** `0x3881`
- "robot":1

```json
{
  "robot":1
}
```
- 当前机器人号；int类型，范围[1,4]
- 控制器回复上位机电机过载保护参数
**命令字：** `0x3882`
- "enable":false
- 电机过载保护使能；bool类型
- "minLimiteTime":10
- 电机过载保护最小确认时间；int类型，范围[10,9999]‰
- "robot":1
- 当前机器人号；int类型，范围[1,4]

### 伺服速度碰撞检测

- 电机扭矩过载值列表；int类型，范围[0,9999)‰或者mNm
- 设置伺服速度碰撞检测参数
**命令字：** `0x40A0`
- "switch":true
- bool类型；速度碰撞检测开
- "speed_delay_time":10
- int类型；伺服速度延迟时间
- "speed_enable_error":10

```json
{
    "switch":true,
    "speed_delay_time":10,
    "speed_enable_error":10
}
```
- double类型；伺服速度允许误差
- 上位机获取伺服速度碰撞检测参数
**命令字：** `0x40A1`
- data：无
- 控制器回复伺服速度碰撞检测参数
**命令字：** `0x40A2`
- "switch":true
- bool类型；速度碰撞检测开
- "speed_delay_time":10
- int类型；伺服速度延迟时间
- "speed_enable_error":10

```json
{
    "switch":true,
    "speed_delay_time":10,
    "speed_enable_error":10
}
```
- double类型；伺服速度允许误差
