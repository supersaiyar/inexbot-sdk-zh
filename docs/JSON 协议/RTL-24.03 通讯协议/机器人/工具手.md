# 工具手

### 工具手参数

- 说明
- 上位机设置工具手参数
**命令字：** `0x3500`
- "parameter":
- 参数列表
- "A":0.0
- 绕A轴旋转；double类型
- 弧度制时范围范围[-3.1416,3.1416]，角度制时范围[-180,180]
- "B":0.0
- 绕B轴旋转；double类型
- 弧度制时范围范围[-3.1416,3.1416]，角度制时范围[-180,180]
- "C":0.0
- 绕C轴旋转；double类型
- 弧度制时范围范围[-3.1416,3.1416]，角度制时范围[-180,180]
- "X":-68.0
- X轴方向偏移；double类型，范围[-2000,2000]mm
- "Y":3.0
- Y轴方向偏移；double类型，范围[-2000,2000]mm
- "Z":162.0
- Z轴方向偏移；double类型，范围[-2000,2000]mm
- "note":""
- 工具手注释；string类型
- "payloadInertia":0.0
- 负载惯量；double类型，范围[-999999,999999]0.001kgm^2
- "payloadMass":3.0
- 负载质量；double类型，范围[0,999999]kg
- "payloadMassCenterX":0.0
- 负载质心X；double类型，范围[-999999,999999]m
- "payloadMassCenterY":0.0
- 负载质心Y；double类型，范围[-999999,999999]m
- "payloadMassCenterZ":0.170
- 负载质心Z；double类型，范围[-999999,999999]m
- "robot":1
- 当前机器人号；int类型，范围[1,4]
- "tool":1

### 工具号

- 当前工具手号；int类型，范围[1,999]
- 说明
- 上位机请求切换工具号
**命令字：** `0x3510`
- "robot":1
- 当前机器人号；int类型，范围[1,4]
- "tool":1

```json
{
  "robot":1,
  "tool":0
}
```
- 需要切换的工具手号；int类型，范围[0,999]
- 0表示无工具手
- 上位机查询工具手号
**命令字：** `0x3511`
- "robot":1

```json
{
  "robot":1
}
```
- 当前机器人号；int类型，范围[1,4]
- 控制器回复上位机工具手号
**命令字：** `0x3512`
- "curToolNum":0
- 当前已选中工具手号；int类型，范围[0,999]
- "robot":1
- 当前机器人号；int类型，范围[1,4]
- "tool":1

```json
{
  "curToolNum":0,
  "robot":1,
  "tool":1
}
```
### 工具标定

- 当前工具手号；int类型，范围[1,999]
- 与已选中工具手号一致，选中工具号为0时值为1
- 说明
- 六点法和七点法标定，包含有标定N号点、获取标定点位置信息、点位标定状态
- 标定计算前需标定好所有点，先用0x3521标定每一个点，所有的点位标定完成后用0x3520计算标定结果
- 查询N号点位置信息时需N号点已标定
- 上位机标定计算请求
**命令字：** `0x3520`
- "robot": 1
- 当前机器人号；int 类型，范围 [1,4]
- "tool":1
- 当前工具手编号；int 类型，范围[1,999]
- "calibrationType":7

```json
{
  "robot": 1,
  "tool": 1,
  "calibrationType": 7
}
```
- 标定点方法；int类型，范围[6,7]
- 6为六点法，7为七点法
- 上位机标定N号点请求
**命令字：** `0x3521`
- "calibrationType":7
- 标定点方法；int类型，范围[6,7]
- 6为六点法，7为七点法
- "point":1
- 标定点编号；int类型
- 六点法时范围[0,5]，七点法时范围[0,6]
- "robot": 1
- 当前机器人号；int 类型，范围 [1,4]
- "tool":1

```json
{
  "calibrationType":7,
  "point":4,
  "robot":1,
  "tool":1
}
```
- 当前工具手编号；int 类型，范围[1,999]
- 控制器回复上位机N号点标定信息
**命令字：** `0x3522`
- "calibrationType":7
- 标定点方法；int类型，范围[6,7]
- 6为六点法，7为七点法
- "point":1
- 标定点编号；int类型
- 六点法时范围[0,5]，七点法时范围[0,6]
- "position":[19998.0,0.000000000001,19998.0,3.14159265359,0.0,0.0,0.0]
- 标定点直角坐标信息列表；double类型
- "robot": 1

```json
{
  "calibrationType":7,
  "point":1,
  "position":[19998.0,0.000000000001,19998.0,3.14159265359,0.0,0.0,0.0],
  "robot":1
}
```
- 当前机器人号；int 类型，范围 [1,4]
- 控制器回复上位机标定计算完成结果
**命令字：** `0x3523`
- "result":true
- 标定计算结果；bool 类型
- "robot": 1
