# 独立轴

### 独立轴参数

- 设置独立轴界面参数
**命令字：** `0x2800`
- "independentAxisParameter":{}
- "axisNum":1
- 轴编号；int类型
- "angularDistanceConversionRatioValue":0.5
- 转换比；double类型
- "isTrack":true
- 是否是地轨；bool类型
- "encoderBitLineditValue":2.0
- 编码器位数；double类型
- "inverseLimitValue":5.0
- 反限位；double类型
- "maximumPositiveSpeedValue":50.0
- 最大正转速；double类型；单位:倍率
- "maximumReverseSpeedValue":50.0
- 最大反转速；double类型；单位:倍率
- "motorDir":1
- 电机方向；int类型；
- 1：正 -1：负
- "positiveLimitValue":0.5
- 正限位；double类型
- "ratedPositiveSpeedValue":2.5
- 额定正转速；double类型；单位:rpm
- "ratedReverseSpeedValue":2.0
- 额定反转速；double类型；单位:rpm
- "reductionRatioValue":2.0
- 减速比；double类型
- "maxAccValue":5.0
- 最大加速度；double类型；单位:速度的倍率
- "maxDecValue":5.0
- 最大减速度；double类型；单位:速度的倍率
- "selectBindServo":2

```json
{
  "independentAxisParameter":
  {
    "axisNum": 0,
    "angularDistanceConversionRatioValue": 0.0,
    "isTrack": false,
    "encoderBitLineditValue": 0.0,
    "inverseLimitValue": 0.0,
    "maximumPositiveSpeedValue": 0.0,
    "maximumReverseSpeedValue": 0.0,
    "motorDir": 0,
    "positiveLimitValue": 0.0,
    "ratedPositiveSpeedValue": 0.0,
    "ratedReverseSpeedValue": 0.0,
    "reductionRatioValue": 0.0,
    "maxAccValue": 0.0,
    "maxDecValue": 0.0,
    "selectBindServo": 0
  }
}
```
- 选择需要绑定的伺服编号；uint类型
- 上位机查询独立轴界面参数
**命令字：** `0x2801`
- "axisNum":1

```json
{
  "axisNum":1
}
```
- 轴编号；int类型
- 控制器返回独立轴界面参数
**命令字：** `0x2802`
- 同0x2800
- 上位机查询独立轴总数
**命令字：** `0x2803`
- data：无
- 控制器返回独立轴总数
**命令字：** `0x2804`
- "axisSum":2

```json
{
  "axisSum":2
}
```
- 独立轴总数
- 上位机删除某一个独立轴参数
**命令字：** `0x2805`
- "axisNum":2

```json
{
  "axisNum":2
}
```
### 独立轴位置

- 轴编号；int类型
- 上位机查询独立轴位置
**命令字：** `0x2806`
- "axisNum":2

```json
{
  "axisNum":2
}
```
- 轴编号；int类型
- 控制器返回独立轴位置
**命令字：** `0x2807`
- "currentPos":210.01

```json
{
  "currentPos":210.01
}
```
- 当前位置；double类型
- 独立轴0点标定
**命令字：** `0x2808`
- "axisNum":2

```json
{
  "axisNum":2
}
```
### 轴独立控制弹窗

- 轴编号
- 独立控制轴PV运动
**命令字：** `0x30E0`
- "robot":1
- 机器人编号；int类型；取值范围[1,4]
- "axisNum":1
- 轴编号；int类型
- "vel":50
- 轴速度；int类型；取值范围[-6000,6000]；单位rpm
- "dir":1
- 运动方向；int类型；
- 1：正；-1：反向
- "acc":50.0
- 加速度；double类型
- "dec":50.0

```json
{
  "robot":1,
  "axisNum":1,
  "vel":50,
  "dir":1,
  "acc":50.0,
  "dec":50.0
}
```
- 减速度；double类型
- 独立控制轴PV停止
**命令字：** `0x30E1`
- "robot":1
- 机器人编号；int类型；取值范围[1,4]
- "axisNum":1

```json
{
  "robot":1,
  "axisNum":1
}
```
- 轴编号；int类型
- 取消轴独立控制运动
**命令字：** `0x30E4`
- "robot":1
- 机器人编号；int类型
- "axisNum":1

```json
{
  "robot":1,
  "axisNum":1
}
```
- 轴编号；int类型
- 独立轴回零
**命令字：** `0x30E5`
- "axisNum":2

```json
{
  "axisNum":2
}
```
- 独立轴编号；int类型
- 独立轴回零停止
**命令字：** `0x30E6`
- "axisNum":2

```json
{
  "axisNum":2
}
```
- 独立轴编号；int类型
- 独立轴点动
**命令字：** `0x30E7`
- "axisNum":2
- 独立轴编号；int类型
- "vel":5
- 速度；int类型；单位:rpm
- "acc":50.0
- 加速度；double类型
- "dec":50.0
- 减速度；double类型
- "dir":1

```json
{
  "axisNum":2,
  "vel":5,
  "acc":50.0,
  "dec":50.0,
  "dir":1
}
```
- 运动方向；int类型；
- 1：正；-1：反向
