# 干涉区

### 干涉区范围

- 说明
- 上位机设置干涉区范围
**命令字：** `0x3800`
- "num":1
- 工艺号；int类型，范围[1,99]
- "param":
- 干涉区参数
- "varName":GB001
- 用于干涉区输出值；string类型；范围GB001-GB999
- "enable":false
- 干涉区使能；bool类型
- "isInInterfer":1
- 在区域内/外；int类型，范围[0,1]
- 0表示在区域外，1表示在区域内
- "maxX":"1101.05"
- 最大X；double类型，范围[-9999,9999]mm
- "maxY":"183.569"
- 最大Y；double类型，范围[-9999,9999]mm
- "maxZ":"1024.13"
- 最大Z；double类型，范围[-9999,9999]mm
- "minX":"833.956"
- 最小X；double类型，范围[-9999,9999]mm
- "minY":"-173.017"
- 最小Y；double类型，范围[-9999,9999]mm
- "minZ":"944.358"
- 最小Z；double类型，范围[-9999,9999]mm
- "port":0
- 输出IO端口；int类型
- "value":0
- 输出参数；int类型，范围[0,1]
- "workType":0
- 起作用方式；int类型，范围[0,1]
- 0表示状态输出，1表示禁止运动
- "robot":1

```json
{
  "num":1,
  "param":
  {
    "enable":false,
    "isInInterfer":1,
    "maxX":"1101.05",
    "maxY":"183.569",
    "maxZ":"1024.13",
    "minX":"833.956",
    "minY":"-173.017",
    "minZ":"944.358",
    "port":0,
    "value":0,
    "workType":0
  },
  "robot":1
}
```
- 当前机器人号；int类型，范围[1,4]
- 上位机查询干涉区范围
**命令字：** `0x3801`
- "num":1
- 当前查询的工艺号；int类型，范围[1,99]
- "robot":1

```json
{
  "num":1,
  "robot":1
}
```
- 当前机器人号；int类型，范围[1,4]
- 控制器回复上位机干涉区参数/干涉点标记结果
**命令字：** `0x3802`
- "num":1
- 工艺号；int类型，范围[1,99]
- "param":
- 干涉区参数
- "calibrationState1":0
- 标记点1是否标定；int类型，范围[0,1]
- 0表示未标定，1表示已标定
- "calibrationState2":0
- 标记点2是否标定；int类型，范围[0,1]
- 0表示未标定，1表示已标定
- "enable":false
- 干涉区使能；bool类型
- "isInInterfer":1
- 在区域内/外；int类型，范围[0,1]
- 0表示在区域外，1表示在区域内
- "maxX":"1101.05"
- 最大X；double类型，范围[-9999,9999]mm
- "maxY":"183.569"
- 最大Y；double类型，范围[-9999,9999]mm
- "maxZ":"1024.13"
- 最大Z；double类型，范围[-9999,9999]mm
- "minX":"833.956"
- 最小X；double类型，范围[-9999,9999]mm
- "minY":"-173.017"
- 最小Y；double类型，范围[-9999,9999]mm
- "minZ":"944.358"
- 最小Z；double类型，范围[-9999,9999]mm
- "point1Data":[0,0,0,0,0,0]
- 标记点1标记位置信息列表；double类型
- "point2Data":[0,0,0,0,0,0]
- 标记点2标记位置信息列表；double类型
- "port":0
- 输出IO端口；int类型
- "value":0
- 输出参数；int类型，范围[0,1]
- "workType":0
- 起作用方式；int类型，范围[0,1]
- 0表示状态输出，1表示禁止运动
- "robot":1

  {

### 干涉点

- 当前机器人号；int类型，范围[1,4]
- 说明
- 上位机请求标记干涉点
**命令字：** `0x3810`
- "calibrationState1":1
- 标记点1状态；int类型，范围[0,1]
- 0表示进行标定，1表示取消标定
- "calibrationState2":1
- 标记点1状态；int类型，范围[0,1]
- 0表示进行标定，1表示取消标定
- "num":1
- 工艺号；int类型，范围[1,99]
- "pos":1
- 标记点号；int类型，范围[1,2]
- 1表示标记点1，2表示标记点2
- "robot":1

```json
{
  "calibrationState1":1,
  "calibrationState2":1,
  "num":1,
  "pos":1,
  "robot":1
}
```
- 当前机器人号；int类型，范围[1,4]
- 上位机请求标记完成计算
**命令字：** `0x3811`
- "num":1
- 当前查询的工艺号；int类型，范围[1,99]
- "robot":1

```json
{
  "num":1,
  "robot":1
}
```
- 当前机器人号；int类型，范围[1,4]
- 上位机请求移动到标记点
**命令字：** `0x3812`
- "num":1
- 当前查询的工艺号；int类型，范围[1,99]
- "pos":1
- 需要移动到的标记点号；int类型，范围[1,2]
- "robot":1

```json
{
  "num":1,
  "pos":1,
  "robot":1
}
```
- 当前机器人号；int类型，范围[1,4]
