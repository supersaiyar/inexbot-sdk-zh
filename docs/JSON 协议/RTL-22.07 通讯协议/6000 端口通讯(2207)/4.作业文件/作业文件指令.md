# 作业文件指令

#### 删除作业文件时，发送：

*0x3111 JOBFILE_DELETE

```json
{
"robot":1,
"jobfilename":"Q"		//作业文件名称
"suffixname":".JBR"	//作业文件后缀名
}
```

#### 重命名作业文件时，发送：

*0x3112 JOBFILE_RENAME

```json
{
"robot":1
"lastname":"Q"		//修改前文件名
"newname":"W"		//修改后文件名
"suffixname":".JBR"	//文件后缀名
}
```

#### 新建作业文件

*0x3113 JOBFILE_CREATE

```json
{
"robot":1
"jobName":"xxxx"			//新建作业文件名
"suffixname":".JBR",		//作业文件后缀名
"time":"JAN 24 18:32:12:2018"	// std::localtime
}
```

#### 打开作业文件

*0x3114 JOBFILE_OPEN

```json
{
"robot":1
"jobName":"xxxx" //作业文件名
"suffixname": ".JBR"   //作业文件后缀名
}
```

#### 复制作业文件

*0x3115 JOBFILE_COPY

```json
{
"robot":1
"oldjobName":"xxxx"   //旧作业文件名
"newjobName":"xxxx" //复制后新文件名
"suffixname":".JBR" //作业文件后缀名
}
```

#### 清空当前机器人作业文件

*0x3116 JOBFILE_CLEAR

```json
{
"robot":1
}
```

#### 设置作业文件为主程序

*0x3117  JOBFILE_MAINJOB_SET

```json
{
"robot":1,
"jobName":"xxx"	//空表示没有
}
```

*0x3118  JOBFILE_MAINJOB_INQUIRE

```json
{
"robot":1,
}
```

*0x3119  JOBFILE_MAINJOB_RESPOND

```json
{
"robot":1,
"jobName":"xxx"	//空表示没有
}
```

#### 保存作业文件

*0x3120  JOB_SAVE

```json
{
"robot":1,        //机器人编号
"jobname":"xxx"	//文件名
"suffix":".JBR"	//文件后缀名
"time":"JAN 24 18:32:12:2018"    //保存时间
}
```

### 程序运行状态

#### 上位机查询程序运行状态时，发送：

*0x3D02 PROGRAMRUN_STATUS_INQUIRE

```json
{
"robot":1
}
```
> 注："robot":	表示机器人1，2，3，4

#### 控制器回复：

*0x3D03  PROGRAMRUN_STATUS_RESPOND

- 命令字：0x2304 DEADMAN_MODE_SET

```json
{
"robot":1,			//机器人号
"status":0			//程序运行状态 0-停止 1-暂停 2-运行
}
```
> 注："robot":	表示机器人1，2，3，4

"status":   程序运行状态： 0-停止 1-暂停 2-运行
