# 自启动程序运行

## 示教器设置

**寄存器：** `0x5081` `SELF_START_RUN_SET`

### 参数说明

| 参数名  | 类型   | 必填 | 说明         | 示例值 |
|---------|--------|------|--------------|--------|
| robot   | int    | 是   | 机器人号     | 1      |
| enable  | bool   | 是   | 是否使能     | false  |
| count   | int    | 是   | 运行次数     | 1      |
| speed   | int    | 是   | 运行速度     | 15     |
| jobname | string | 是   | 作业文件名字 | "Q1"   |

### JSON 示例

```json
{
    "robot": 1,
    "enable": false,
    "count": 1,
    "speed": 15,
    "jobname": "Q1"
}
```
