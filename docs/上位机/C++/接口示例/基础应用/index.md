# 基础应用

基础使用示例，适合刚接触 C++ SDK 的用户。

## 示例列表

| # | 文件 | 说明 |
|---|------|------|
| 1 | 1.获取不同坐标系的位置.md | 查询关节、直角、工具、用户坐标系位置 |
| 2 | 2.不考虑平滑的单点运动.md | 示教模式 + 上电 + robot_movej 单轴移动 |
| 3 | 3.平滑的连续轨迹运动.md | 运动队列模式，队列下发 + 平滑过渡 |
| 4 | 4.示教模式类型切换与轨迹回放功能.md | 拖拽示教、轨迹记录与回放 |
| 5 | 5.工具手标定.md | 7 点标定 / 20 点标定完整流程 |
| 6 | 6.运动队列的曲线运动.md | MOVS 曲线插补，10 个点位连续轨迹 |

## 前提说明

控制器有三种运行模式：示教模式、运行模式、远程模式。示教模式下运动需要上电操作；运动模式只需打开运动模式、确保伺服就绪即可。
