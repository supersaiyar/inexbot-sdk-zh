# 纳博特二次开发文档库 (NexDroid SDK Docs)

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](LICENSE)
[![Version](https://img.shields.io/badge/Version-2026.03-green.svg)]()
[![Website](https://img.shields.io/badge/Website-inexbot.com-orange.svg)](https://www.inexbot.com)

> **运动繁复，化简为境 (Simplify Complexity, Master Motion)**

纳博特（NexDroid）是国内领先的工业机器人控制器厂商，自主研发开放型软件平台，提供从底层硬件到上层应用的全栈式二次开发接口（NexDroidAPI）。本文档库面向使用纳博特机器人控制系统的二次开发用户，包含完整的 API 参考、开发指南和示例代码。

---

## 📖 文档总览

| 开发层级 | 说明 | 开发语言 |
|----------|------|----------|
| **控制器二次开发** | 直接在控制器内运行程序 | Linux, C++ |
| **示教器二次开发** | 定制专属 UI 界面和功能 | Linux, Qt/C++ |
| **上位机 SDK** | 通过 API 直接控制机器人 | Windows/Linux, C++/C#/Python |
| **JSON 协议** | 基于 JSON 格式的 TCP 通讯 | 任意语言 |
| **ROS 开发** | ROS 机器人操作系统集成 | Linux, C++/Python |
| **HAL 层开发** | 硬件抽象层，访问底层硬件 | Linux, C++ |
| **EtherCAT 主站** | 工业总线主站开发 | Linux, C++ |

## 🚀 推荐学习路径

### 上位机控制（推荐入门）

适合需要快速集成机器人控制的工程师：

1. 阅读 **02-入门指南.md** 选择开发形式
2. 学习 **05-上位机SDK/** 中对应语言的开发指南
3. 参考示例代码进行实践

### 工艺定制开发

适合需要定制焊接、码垛等工艺的用户：

1. 掌握上位机基础控制
2. 学习 **08-JSON协议参考/04-工艺应用/**
3. 结合控制器二次开发进行深度定制

### 科研与 ROS 集成

适合科研用户、ROS 开发者：

1. 学习 **07-ROS开发/** 中的安装教程
2. 参考 ROS 示例进行机器人控制
3. 结合 JSON 协议进行高级功能开发

## 📦 技术规格

| 项目 | 规格 |
|------|------|
| 机器人数量 | 支持最多 4 台机器人 |
| 坐标系类型 | 关节坐标、直角坐标、工具坐标、用户坐标、大地坐标 |
| 通讯端口 | 5000（文件传输）、6000（命令控制）、7000（状态广播） |
| 外部轴 | 支持最多 5 个外部轴 |
| 推荐操作系统 | Ubuntu 20.04 LTS |
| 推荐编译器 | GCC 9.4.0 或更高 |
